## 拿Java代码说话：智谱清言接入示例，一个字不改你也跑得通

很多Java兄弟最烦的事，是看一堆概念，不如给张代码截图。今天我就把核心封装拿出来。这是基于Spring Boot + OkHttp的示例，你用你的OpenAI旧项目，把base_url和api_key改掉，直接就跑。

java
import okhttp3.*;
import com.fasterxml.jackson.databind.ObjectMapper;
import java.util.*;

public class ZhiPuAIClient {
    private static final String BASE_URL = "https://www.yunwuai.cc/v1/chat/completions";
    private static final String API_KEY = "sk-your-yunwuyunwuai-key"; // 替换成你在云雾API申请的key
    private static final MediaType JSON = MediaType.get("application/json; charset=utf-8");

    private final OkHttpClient client = new OkHttpClient();
    private final ObjectMapper mapper = new ObjectMapper();

    public String callGLM4(String userMessage) throws Exception {
        // 构建请求体，模型名称根据智谱官方OpenAI兼容接口填写：glm-4
        Map<String, Object> requestBodyMap = new HashMap<>();
        requestBodyMap.put("model", "glm-4");
        requestBodyMap.put("messages", List.of(
            Map.of("role", "system", "content", "你是一个Java工程师助手"),
            Map.of("role", "user", "content", userMessage)
        ));
        requestBodyMap.put("temperature", 0.7);
        requestBodyMap.put("max_tokens", 1024);

        String json = mapper.writeValueAsString(requestBodyMap);
        RequestBody body = RequestBody.create(json, JSON);

        Request request = new Request.Builder()
                .url(BASE_URL)
                .addHeader("Authorization", "Bearer " + API_KEY)
                .post(body)
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (!response.isSuccessful()) {
                throw new RuntimeException("Unexpected code " + response);
            }
            // 解析返回的JSON
            Map<String, Object> responseMap = mapper.readValue(response.body().string(), Map.class);
            List<Map<String, Object>> choices = (List<Map<String, Object>>) responseMap.get("choices");
            Map<String, Object> firstChoice = choices.get(0);
            Map<String, Object> message = (Map<String, Object>) firstChoice.get("message");
            return (String) message.get("content");
        }
    }
}

对于Java项目来说，关键就两点：
- 依赖：`okhttp3` + `jackson-databind`（Maven里加一下就行）
