### 接入有多简单——Java 5行代码示例

真的只是改一行base_url、换一个apiKey的事。下面这个Java示例用OpenAI官方Java SDK（或任意兼容OpenAI的客户端），调用o3-mini模型，全部核心代码就5行：

java
import com.theokanning.openai.service.OpenAiService;
import com.theokanning.openai.completion.chat.*;

// 1. 初始化客户端（修改baseUrl和apiKey）
OpenAiService service = new OpenAiService(
    "https://www.yunwuai.cc/v1",
    "your-yunwu-api-key"
);

// 2. 构造消息
ChatCompletionRequest request = ChatCompletionRequest.builder()
    .model("o3-mini")        // 一行切换模型：o3-mini / gpt-4o / claude-3.5-sonnet
    .messages(List.of(
        new ChatMessage("system", "You are a helpful assistant."),
        new ChatMessage("user", "用Java写一个快速排序算法")
    ))
    .build();

// 3. 发送请求并获取回复
ChatCompletionResult result = service.createChatCompletion(request);
String reply = result.getChoices().get(0).getMessage().getContent();

// 4. 输出结果
System.out.println(reply);
