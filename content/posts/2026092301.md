## 第三步：开启流式输出（Streaming）

上面那段代码关闭了流式输出，适合新手观察完整结果。但如果你想做实时对话、打字机效果，需要打开流式输出。

代码稍微改一下：

python
from openai import OpenAI

client = OpenAI(
    base_url="https://www.yunwuai.cc/v1",
    api_key="<你的云雾 API Key>"
)

response = client.chat.completions.create(
    model="qwen-plus",
    messages=[
        {"role": "user", "content": "用幽默的方式介绍一下 Qwen-Plus 这个 AI 模型"}
    ],
    stream=True  # 重点：开启流式输出
)

for chunk in response:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")


这样跑起来，回复就会一个字一个字地往外蹦，体验就像跟 ChatGPT 聊天一模一样。
