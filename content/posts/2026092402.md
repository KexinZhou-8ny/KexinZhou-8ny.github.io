### 第三步：编写核心调用代码

我们直接用OpenAI SDK创建请求，指定模型名为 `"qwen-vl-plus"` 或 `"qwen-vl-max"`（实测支持）。下面是一个完整的Java方法：

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.completion.chat.ChatMessageRole;
import com.theokanning.openai.image.CreateImageRequest;

import java.time.Duration;
import java.util.ArrayList;
import java.util.List;

public class QwenVLJavaSample {
    public static void main(String[] args) {
        // 设置API Key和baseUrl
        String token = "sk-yunwu-key"; // 替换成你的
        OpenAiService service = new OpenAiService(token, Duration.ofSeconds(30));
        service.setBaseUrl("https://www.yunwuai.cc/v1");

        // 准备用户消息，包含图片URL
        List<ChatMessage> messages = new ArrayList<>();
        String imageUrl = "https://img.yunwuai.cc/demo/shoes.png"; // 你的图片地址
        String contentWithImage = "请描述下面这张图片中的物体，并给出一个电商标题\n![" + imageUrl + "](" + imageUrl + ")";
        
        messages.add(new ChatMessage(ChatMessageRole.USER.value(), contentWithImage));

        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("qwen-vl-plus") // 核心：换成通义千问视觉模型
                .messages(messages)
                .maxTokens(500)
                .temperature(0.7)
                .build();

        // 发送请求并获取结果
        service.createChatCompletion(request).getChoices().forEach(choice -> {
            System.out.println(choice.getMessage().getContent());
        });

        // 关闭service
        service.shutdownExecutor();
    }
}


**关键点**：用`qwen-vl-plus`作为模型名，API地址换成云雾的。图片可以传URL，也可以传Base64编码的字符串（参考阿里官方文档）。这里为了简单，我用URL方式演示。
