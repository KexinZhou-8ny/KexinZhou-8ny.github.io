## 第三步：保姆级 Node.js 代码示例

把下面的代码完整复制进去。这里我特意为你准备了**Llama 4 统一接入**的标准示例，**连代码注释都给你写了**，你只需要做一步修改：把 `API_KEY_HERE` 替换成你刚才复制的密钥。

javascript
// chat.js
// 这个例子展示如何通过[[云雾ai官网](https://www.yunwuai.cc/)](https://www.yunwuai.cc/)的API，统一接入Llama 4模型。

const openai = require('openai'); // 如果你是第一次用这个库，需要先 npm install openai

// 请将 YOUR_API_KEY 替换成你在云雾后台申请的 API Key
const API_KEY = 'YOUR_API_KEY'; // ！！！ 这里改成你的密钥 ！！！
const BASE_URL = 'https://www.yunwuai.cc/v1'; // 云雾的API接口，完全兼容OpenAI格式

const client = new openai.OpenAI({
    apiKey: API_KEY,
    baseURL: BASE_URL,
});

async function callLlama4() {
    try {
        console.log('正在向 Llama 4 发起对话...');
        const stream = await client.chat.completions.create({
            // 重点：这里直接指定 llama-4 模型名称
            model: 'llama-4', 
            messages: [
                { role: 'system', content: '你是一个乐于助人的助手。' },
                { role: 'user', content: '用中文简单介绍一下自己，并列举3个你能解决的问题。' },
            ],
            // 我们开启流式输出，实时看到结果
            stream: true,
        });

        let fullResponse = '';
        for await (const chunk of stream) {
            const content = chunk.choices[0]?.delta?.content || '';
            process.stdout.write(content); // 真正的实时打印
            fullResponse += content;
        }
        console.log('\n\n对话完成。');
    } catch (error) {
        console.error('调用 Llama 4 出错：', error);
    }
}

callLlama4();



**你看懂了吗？** 核心就是改两处：
1.  `baseURL` 设为 `https://www.yunwuai.cc/v1`。
2.  `model` 设为 `'llama-4'`。

**就这么简单。** 完全兼容 OpenAI 的调用方式，你以前用 OpenAI API 写的代码，改个 base_url 和 model 名，直接无缝迁移。
