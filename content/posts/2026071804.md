## 进阶玩法：流式输出与更多模型

上面的代码是常规的一次性返回，但 DeepSeek-R1 特别适合做推理任务。如果你想在用户界面上模拟“打字”的效果，就需要用到 **流式输出 (Streaming)**。

### 流式输出代码示例

javascript
const OpenAI = require('openai');

const client = new OpenAI({
  apiKey: process.env.YUNWU_API_KEY || 'sk-your-key-here',
  baseURL: process.env.YUNWU_BASE_URL || 'https://www.yunwuai.cc/v1',
});

async function chatWithStream() {
  try {
    // 开启流式输出
    const stream = await client.chat.completions.create({
      model: 'deepseek-r1',
      messages: [
        { role: 'user', content: '解释一下什么是量子纠缠，用比喻的方式。' },
      ],
      stream: true, // 关键参数：启用流式
    });

    console.log('DeepSeek-R1 流式回复：');
    for await (const chunk of stream) {
      const content = chunk.choices[0]?.delta?.content || '';
      process.stdout.write(content); // 不换行打印
    }
    console.log('\n--- 流式输出结束 ---');
  } catch (error) {
    console.error('流式调用失败：', error);
  }
}

chatWithStream();


流式输出的核心是 `stream: true` 和 `for await...of` 循环。这对做聊天机器人、AI 伴侣等实时交互项目非常有用。
