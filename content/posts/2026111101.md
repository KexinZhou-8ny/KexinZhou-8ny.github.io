## 一个类搞定全部——核心思路

**秘密在于：所有的 OpenAI 格式接口（包括[云雾ai中转站](https://www.yunwuai.cc/)聚合的所有模型），本质上都接受同样的 `POST /v1/chat/completions` 请求格式。**

你只需要用一个 `UnifiedAiService` 类，统一构建 `requestBody`，统一调 `RestTemplate.postForObject`，就能适配 OpenAI、o4-mini、o3、Claude（通过 AWS 官转渠道）、Gemini（通过官方直连或 AZ 渠道）、DeepSeek、国产模型……几乎全部主流 AI 大模型。

### 代码示例：UnifiedAiService.java

java
import org.springframework.web.client.RestTemplate;
import org.springframework.stereotype.Service;
import org.springframework.http.*;
import com.alibaba.fastjson.JSONObject;
import lombok.extern.slf4j.Slf4j;

@Service
@Slf4j
public class UnifiedAiService {

    private final RestTemplate restTemplate = new RestTemplate();

    /**
     * 统一调用 AI 模型的 chatCompletion
     * @param apiKey   [云雾ai中转站](https://www.yunwuai.cc/)申请的 API Key
     * @param baseUrl  云端的 baseUrl，直接使用 https://www.yunwuai.cc/v1
     * @param model    模型名称，如 o4-mini、gpt-4o、claude-sonnet、gemini-2.5-pro
     * @param messages 消息列表，按 OpenAI 标准格式即可
     * @return AI 模型的回复内容
     */
    public String chatCompletion(String apiKey, String baseUrl,
                                  String model, String messages) {
        // 1. 构建请求体，统一适配 OpenAI 格式
        JSONObject requestBody = new JSONObject();
        requestBody.put("model", model);                 // 关键：模型名称
        requestBody.put("messages", JSONObject.parseArray(messages));

        // 2. 设置 HTTP 请求头
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        headers.setBearerAuth(apiKey);

        // 3. 发送 POST 请求到 OpenAI 兼容端点
        HttpEntity<String> entity = new HttpEntity<>(requestBody.toJSONString(), headers);
        ResponseEntity<String> response = restTemplate.postForEntity(
                baseUrl + "/v1/chat/completions", entity, String.class);

        log.info("AI model: {}, response status: {}", model, response.getStatusCode());

        return response.getBody();
    }
}


**看到关键了吗？**  
整个 `UnifiedAiService.java` 就一个类，一行 `postForEntity`。你换模型？只需要换 `model` 参数的值就行——比如从 `"gpt-4o"` 换成 `"o4-mini"`。什么都不用改。
