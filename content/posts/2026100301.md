### 第二步：编写Java代码

我们需要一个HTTP客户端。这里以OkHttp为例，使用Maven依赖。

1.  **添加Maven依赖** (在`pom.xml`中)：

    <dependency>

2.  **编写核心调用代码** (以Gemini Pro为例)：
    java
    import okhttp3.*;
    import com.google.gson.Gson;
    import com.google.gson.JsonArray;
    import com.google.gson.JsonObject;
    import java.io.IOException;

    public class GeminiDemo {

        // 1. API配置 - 从云雾AI获取
        private static final String API_KEY = "sk-你的云雾APIKey"; // 替换成你的Key
        private static final String BASE_URL = "https://www.yunwuai.cc/v1";
        private static final String MODEL = "gemini-pro"; // 或你想要的Gemini模型名

        private static final OkHttpClient client = new OkHttpClient();
        private static final Gson gson = new Gson();

        public static void main(String[] args) {
            // 2. 构建请求体
            JsonObject message = new JsonObject();
            message.addProperty("role", "user");
            message.addProperty("content", "你好，请用中文介绍下你自己，并说说你是什么模型。");

            JsonArray messages = new JsonArray();
            messages.add(message);

            JsonObject requestBody = new JsonObject();
            requestBody.addProperty("model", MODEL);
            requestBody.add("messages", messages);

            // 3. 构建OkHttp请求
            Request request = new Request.Builder()
                    .url(BASE_URL + "/chat/completions")
                    .addHeader("Authorization", "Bearer " + API_KEY)
                    .post(RequestBody.create(MediaType.parse("application/json; charset=utf-8"), requestBody.toString()))
                    .build();

            // 4. 发送请求并处理响应
            try (Response response = client.newCall(request).execute()) {
                if (!response.isSuccessful()) {
                    System.out.println("请求失败: " + response.code() + " " + response.message());
                    return;
                }

                JsonObject responseJson = gson.fromJson(response.body().string(), JsonObject.class);
                String reply = responseJson
                        .getAsJsonArray("choices")
                        .get(0)
                        .getAsJsonObject()
                        .get("message")
                        .getAsJsonObject()
                        .get("content")
                        .getAsString();

                System.out.println("模型回复: " + reply);
            } catch (IOException e) {
                System.out.println("请求异常: " + e.getMessage());
            }
        }
    }

    

3.  **运行代码**：将你的API Key填入后，直接运行`main`方法。控制台会打印出Gemini模型的回复。整个过程流畅、无延迟，不需要任何特殊的网络配置。
