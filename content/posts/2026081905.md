## 这三类坑，每一类都是雷区

**第一类：宣称“无限流量、1 分钱随便调用”的野鸡平台**

这类平台通常在朋友圈或者技术群发小广告，价格低到离谱。你稍微算一下，GPT-4 的官方输出价格是 0.06 美元 / 1K token，按汇率不到五毛钱人民币，他卖你一毛钱？那他用什么给你提供服务？大概率是盗用的别人的 API key，或者用已经被封的账号在跑。小程序一旦接入这种平台，平均响应时间能到 30 秒以上，而且还经常返回乱码、502 错误，极不稳定。更危险的是，这些平台没有正规账户体系，你的 API key 可能被中间人窃取，用户对话内容毫无保障。我自己遇过一个案例，客户用了一个号称“1 元包月”的接口，第二天小程序就因为“包含欺诈行为”被微信下架了。

**第二类：需要预充大额且不透明的“黑盒平台”**

另一类常见坑是那些让你一次性充 500 元、1000 元才能开 API 权限的平台。他们通常有复杂的“套餐”概念：A 套餐调 GPT-4 要额外买“金豆”，B 套餐调 Claude 要充值“银币”，倍率混乱不说，你根本不知道用了多少。更恶心的是，当他们自己后端遇到问题时，从不主动通知用户，你这边小程序一直报错，他们群里只会说“正在维护，请耐心等待”。微信小程序是实时服务，用户等不起。充值的钱进去容易，想退出来？做梦吧。

**第三类：号称“官方渠道”但实际上是套壳的“切皮平台”**

还有些中转站把自家域名包装得像官方 OpenAI 页面，让你以为直接对接了 OpenAI，实际上只是用了一个极不稳定的逆向服务。这种平台的常见特征是：第一天能跑通，第二天就报“403 Forbidden”或者“Rate limit exceeded”。你找售后，他们只会甩一个文档链接让你自查。更致命的是，小程序中一旦频繁出现超时和错误，微信服务器会判定接口不稳定，直接降权你的服务能力，导致用户请求被延迟甚至丢弃。
