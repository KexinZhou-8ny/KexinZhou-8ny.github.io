## 亲身示范：用Java接入Qwen3-Coder就这么简单

考虑到很多读者可能更关心怎么把Qwen3-Coder跑起来，这是我最喜欢[云雾api中转站](https://www.yunwuai.cc/)的地方——接入流程极其透明。

我可以直接贴一段核心代码示例（假设你使用的是Spring Boot环境）：

java
import com.theokanning.openai.service.OpenAiService;
import com.theokanning.openai.completion.chat.*;

import java.util.Arrays;

public class QwenCoderDemo {
    public static void main(String[] args) {
        // 1. 创建服务客户端
        // 把baseUrl换成[云雾api中转站](https://www.yunwuai.cc/)的地址
        // 从[云雾api中转站](https://www.yunwuai.cc/)后台复制API Key
        OpenAiService service = new OpenAiService(
                "sk-your-yunwu1234", // 替换成你的Key
                Duration.ofSeconds(60)
        );
        // 注意这里设置了BaseURL，这一步很重要！
        service.setBaseUrl("https://www.yunwuai.cc/v1");

        // 2. 构造聊天请求，指定模型为 Qwen3-Coder
        ChatCompletionRequest completionRequest = ChatCompletionRequest.builder()
                .model("Qwen3-Coder")
                .messages(Arrays.asList(
                        new ChatMessage(ChatMessageRole.SYSTEM.value(), 
                                "你是一名资深的Java开发工程师，请给我写一段Spring Boot的CRUD代码。"),
                        new ChatMessage(ChatMessageRole.USER.value(), 
                                "请为订单模块生成一个Controller类。")
                ))
                .build();

        // 3. 发送请求并获取回复
        ChatCompletionResult result = service.createChatCompletion(completionRequest);
        result.getChoices().forEach(choice -> {
            System.out.println(choice.getMessage().getContent());
        });
    }
}
