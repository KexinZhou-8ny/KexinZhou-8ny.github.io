快速排序（Quicksort）是一种高效的排序算法，它采用分治策略。

1. **选择基准（Pivot）**：从数组中选一个元素作为基准。
2. **分区（Partition）**：将比基准小的元素移到基准左边，大的移到右边。
3. **递归排序**：递归地对左右两个子数组重复步骤1和2。

Python实现如下：

python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

# 示例
arr = [3, 6, 8, 10, 1, 2, 1]
print(quicksort(arr))
# 输出: [1, 1, 2, 3, 6, 8, 10]


**解释**：
- 我们选择中间元素作为基准。
- 使用列表推导式将数组分为三部分。
- 递归排序左右子数组，最后合并。
- 这是一种简洁的原地排序实现，平均时间复杂度为O(n log n)。

如果你希望模型像聊天一样一个字一个字地生成答案，而不是一次性输出完整文本，可以用`stream=True`参数。这种方式在实时交互中体验更好，尤其适合写代码、写长文等场景。

python

    model="llama3-70b",
    messages=[{"role": "user", "content": "为一个小型企业写一份20字的slogan"}],
    stream=True
)
for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")
