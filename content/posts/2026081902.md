## Java接入有多简单——真的只需要3分钟

这里我直接贴一个完整的示范。假设你的项目里已经引入了openai-java-sdk（或类似OpenAI兼容的HTTP客户端）：

### 第一步：引入Maven依赖（如果你用Spring Boot）

xml
<dependency>
    <groupId>com.theokanning.openai-gpt3-java</groupId>
    <artifactId>service</artifactId>
    <version>0.18.2</version>
</dependency>


### 第二步：修改基础URL和API Key

你不需要任何额外的阿里、百度或腾讯SDK。直接在配置文件里改：

java
// 你的Spring Boot application.yml
openai:
  base-url: https://www.yunwuai.cc/v1
  api-key: 从云雾平台申请的key
  model: ernie-4.0 # 想换模型时，把这行改成 "qwen-max" 或 "hunyuan-pro"


### 第三步：编写调用逻辑

java
@Service
public class UnifiedModelService {
    
    private final OpenAiService openAiService;
    
    public UnifiedModelService() {
        // 初始化客户端，连接[云雾api聚合平台](https://www.yunwuai.cc/)
        openAiService = new OpenAiService("你的云雾API Key", Duration.ofSeconds(30));
        // 设置自定义baseUrl
        openAiService.setBaseUrl("https://www.yunwuai.cc/v1");
    }
    
    public String callModel(String modelName, String userMessage) {
        // 构造请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model(modelName)  // 传 "ernie-4.0" 或 "qwen-max" 或 "hunyuan-standard"
                .messages(List.of(new ChatMessage(ChatMessageRole.USER.value(), userMessage)))
                .maxTokens(1000)
                .build();
        
        // 发起请求
        ChatCompletionResult result = openAiService.createChatCompletion(request);
        
        // 取出回复
        return result.getChoices().get(0).getMessage().getContent();
    }
}


### 第四步：一行代码完成模型切换

java

// 用混元
String reply3 = modelService.callModel("hunyuan-pro", "请用中文回答：今天天气如何？");


三行代码，调用三个完全不同的厂商模型。从配置到跑通，三分钟绰绰有余。你的LangChain、Spring AI、或自定义HTTP客户端，基本不需要改任何其他东西。
